
# First Human City Layout — Minimal Version

This city is side-view-friendly: wide left-to-right route with upper rooftops and lower sewer path.

```text
[Darkness Cave]
   |
   v
B02 Old Drain Culvert ---- B21 Back Alley ---- B23 Sewer Run ---- B25 Return Tunnel
   |                              |                    |
   v                              v                    v
B01 North Gate -> B03 Well -> B04 Market -> B05 Bakery -> B06 Forge -> B20 Bridge -> B09 Barracks -> B12 Chapel -> B13 Cemetery -> B18 Mayor -> B17 Storehouse
                                               |                         |                         |
                                               v                         v                         v
                                            B08 Stable               B22 Rooftops             B11 East Tower
                                                                  B10 West Tower
```

## Build Rule
Start with one horizontal main street, one upper rooftop lane, one lower sewer/back-alley lane. Do not make a top-down city yet.

## Gameplay Rule
A direct run along the main street should usually fail. A planned route through timing, distractions, rooftop/sewer lanes, or citizen schedule gaps should succeed.

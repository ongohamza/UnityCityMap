
# GIMP Cleanup Checklist

For each townsperson or building asset:

1. Open source image.
2. Crop to one subject.
3. Add alpha channel.
4. Remove background.
5. Clean silhouette with Pencil, not Brush.
6. Remove extra limbs, broken weapons, mutated shapes.
7. Resize only with Interpolation: None.
8. Export PNG, not JPG.
9. Name file clearly: citizen_role_name_v1.png.
10. Import to Unity with Sprite / Point / Compression None.

# Human City QA Checklist

[ ] City scene opens.
[ ] Main street exists.
[ ] Rooftop/upper route exists.
[ ] Sewer/back-alley route exists.
[ ] 25 location waypoints exist.
[ ] 50 citizens spawn or can be manually placed.
[ ] Each citizen has 3-5 schedule blocks.
[ ] City clock advances.
[ ] Citizens move to correct locations.
[ ] At least 5 civilians flee or raise alarm when player is seen.
[ ] At least 5 guards pursue or attack when full alarm starts.
[ ] Bell/alarm node raises city danger.
[ ] Route choice triggers raise alarm according to risk.
[ ] Distraction trigger creates a temporary safer timing window.
[ ] Direct rush through main street usually fails.
[ ] Sewer route can reach storehouse objective.
[ ] Rooftop route can reach storehouse objective.
[ ] Market/gatehouse timing route can reach storehouse objective.
[ ] Storehouse loot can be stolen.
[ ] Return tunnel/escape only completes after loot is collected.
[ ] Full alarm visibly changes citizen behavior.
[ ] Build runs outside Unity.

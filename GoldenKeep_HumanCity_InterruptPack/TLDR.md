# TLDR

Pause Phase 3 at card 38 and build a minimal first human city with real systemic behavior.

- 50 citizens have names, roles, personalities, alarm/combat behavior, and schedules.
- 25 city locations define the city network.
- Citizens move through waypoints using daily schedule blocks.
- The player observes patterns, chooses route nodes, steals storehouse loot, and escapes.
- Direct rushing through the main street should usually trigger full alarm and fail.
- Planned sewer, rooftop, market distraction, and gatehouse timing routes should succeed.
- The image-to-3D pipeline is not part of this pack anymore.
- First implementation uses 2D stand-ins or a tiny batch of finished model outputs.

This is not "make a big city." It is "make the smallest city that already behaves like a living system."
